# File2QR

JS-powered npm tool to convert files to QR codes images.

# Installation

`npm install file2qr`

# CLI Tool

`file2qr <input-file> [output-file]`

# In-Code Usage

```javascript
const { htmlToQr } = require('your-package-name');
// Example usage
htmlToQr('path/to/input.html', 'path/to/output.png');
```
